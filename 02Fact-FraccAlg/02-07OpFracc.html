Operaciones con Fracciones Algebraicas

En esta sección aprenderás a realizar operaciones básicas con fracciones algebraicas de manera paso a paso.

Suma de fracciones algebraicas con el mismo denominador

Problema: Sumemos:

$$\frac{6}{2x^2 - 7} + \frac{-3x - 8}{2x^2 - 7}$$

Paso 1: Como ambas fracciones tienen el mismo denominador, conservamos el denominador y sumamos los numeradores:

$$\frac{6 + (-3x - 8)}{2x^2 - 7}$$

Paso 2: Simplificamos el numerador:

$$6 + (-3x - 8) = 6 - 3x - 8 = -2 - 3x$$

Resultado final:

$$\frac{6}{2x^2 - 7} + \frac{-3x - 8}{2x^2 - 7} = \frac{-2 - 3x}{2x^2 - 7}$$

Resta de fracciones algebraicas con el mismo denominador

Problema: Restemos:

$$\frac{9x^2 + 3}{14x^2 - 9} - \frac{-3x^2 + 5}{14x^2 - 9}$$

Paso 1: El denominador es el mismo, conservamos el denominador y restamos los numeradores:

$$\frac{(9x^2 + 3) - (-3x^2 + 5)}{14x^2 - 9}$$

Paso 2: Distribuimos el signo negativo:

$$(9x^2 + 3) - (-3x^2 + 5) = 9x^2 + 3 + 3x^2 - 5$$

Paso 3: Agrupamos términos semejantes:

$$= (9x^2 + 3x^2) + (3 - 5) = 12x^2 - 2$$

Resultado final:

$$\frac{9x^2 + 3}{14x^2 - 9} - \frac{-3x^2 + 5}{14x^2 - 9} = \frac{12x^2 - 2}{14x^2 - 9}$$

Suma de fracciones algebraicas con diferente denominador

Problema: Sumemos:

$$\frac{-5x}{8x + 7} + \frac{-6x^3}{3x + 1}$$

Paso 1: Determinamos el mínimo común denominador:

$$(8x + 7)(3x + 1)$$

Paso 2: Reescribimos ambas fracciones con el denominador común:

$$\frac{-5x(3x + 1)}{(8x + 7)(3x + 1)} + \frac{(8x + 7)(-6x^3)}{(8x + 7)(3x + 1)}$$

Paso 3: Desarrollamos los numeradores:

$$-5x(3x + 1) = -15x^2 - 5x$$ $$(8x + 7)(-6x^3) = -48x^4 - 42x^3$$

Paso 4: Sumamos los numeradores:

$$(-15x^2 - 5x) + (-48x^4 - 42x^3) = -48x^4 - 42x^3 - 15x^2 - 5x$$

Resultado final:

$$\frac{-5x}{8x + 7} + \frac{-6x^3}{3x + 1} = \frac{-48x^4 - 42x^3 - 15x^2 - 5x}{(8x + 7)(3x + 1)}$$

Resta de fracciones algebraicas con denominadores factorizables

Problema: Restemos:

$$\frac{a - 2}{a + 2} - \frac{a - 3}{a^2 + 4a + 4}$$

Paso 1: Factorizamos el segundo denominador:

$$a^2 + 4a + 4 = (a + 2)^2$$

Paso 2: Determinamos el mínimo común denominador:

$$(a + 2)^2$$

Paso 3: Reescribimos ambas fracciones con el denominador común:

$$\frac{(a - 2)(a + 2)}{(a + 2)^2} - \frac{a - 3}{(a + 2)^2}$$

Paso 4: Desarrollamos el numerador de la primera fracción:

$$(a - 2)(a + 2) = a^2 - 4$$

Paso 5: Realizamos la resta de numeradores:

$$(a^2 - 4) - (a - 3) = a^2 - 4 - a + 3 = a^2 - a - 1$$

Resultado final:

$$\frac{a - 2}{a + 2} - \frac{a - 3}{a^2 + 4a + 4} = \frac{a^2 - a - 1}{(a + 2)^2}$$
